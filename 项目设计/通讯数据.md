# 1.项目通信数据格式

服务端返回的数据格式
```json
{
   "code":"1", // 0代表操作正常，非0代表出错
   "msg":"reMsg" //返回的消息
   "data":{
        "user":{
           "account":"myname",
           "age":18,
           "password":"xxx"
        }
   }
}
```

客户端传输的格式
```json
{
  "user":{
             "account":"myname",
             "age":18,
             "password":"xxx"
   }
}
```

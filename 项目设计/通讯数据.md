# 1.项目通信数据格式

```json
{
   "code":"1", // 0代表操作正常，非0代表出错
   "msg":"reMsg", //返回的消息
    "user":{
         "uno":"myname",
         "name":"xx",
         "pwd":"xxx",
         "birthday":"xxx",
         "sex":"xxx",
         "idcard":"xxx",
         "phone":"xxx",
         "intro":"xxx",
         "title":"xxx",
         "role":"xxx"
    }
   
}
```

客户端传输的格式
```json
{

   "account":"myname",
   "pwd":"xxx"
   
}
```
